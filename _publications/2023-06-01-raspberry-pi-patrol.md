**Role:** First author.

An early systems project: a **Raspberry-Pi-based security patrol robot** designed for complex construction sites.

- Integrates **STM32** with multiple sensors for environment perception.
- Uses **Kalman filtering** for state estimation.
- Uses **FAST-RCNN** for hazard-region recognition.
- Supports **autonomous localization, obstacle avoidance, and target recognition**.

The system can replace part of the manual patrol work, improving automation and real-time safety management on construction sites.
